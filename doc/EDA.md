#### 質的変数

- sex char 性別
- embarked char 乗船した港（S=Southampton, C=Cherbourg, Q=Queenstown）

- survived boolean 生還結果（1=生還, 0=死亡）
- age int 年齢
- sibsp int 乗船していた兄弟、配偶者の数
- parch int 乗船していた両親、子供の数
- fare float 運賃
